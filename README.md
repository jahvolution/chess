# Chess Game

## Quickstart

### 1. Download & Install
- Clone the repository:
  ```sh
  git clone https://github.com/yourusername/chess-game.git
  cd chess-game

- Install dependencies:
  `npm install`

### 2. Run the Game
- Start the game with:
`npm start`

### 3. Play
- Run the following commands:
```sh
rails c
game = Game.new
game.play
